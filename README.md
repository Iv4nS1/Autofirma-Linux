# Guía Completa: AutoFirma en Linux y Firmas Profesionales

Bienvenido a este repositorio. Aquí encontrarás la solución definitiva para instalar, configurar y utilizar AutoFirma en distribuciones Linux sin frustraciones. Además, aprenderás a estampar tu firma de forma visual y profesional en cualquier documento PDF.

El uso de certificados digitales en Linux puede ser un desafío, especialmente en entornos modernos con Wayland y las últimas versiones de navegadores. Esta guía está diseñada para resolver esos problemas comunes.

Este repositorio se compone de dos partes principales:

1. [Configuración técnica de AutoFirma en navegadores (Firefox / Google Chrome)](CONFIG-AUTOFIRMA.md)
2. [Guía para generar y estampar una firma visible profesional](firma.md)

**Requisitos Generales:**
* Una distribución Linux (guía centrada en Fedora/Red Hat, pero adaptable a Ubuntu/Debian).
* Tu certificado digital ya instalado en tu navegador.
* Ganas de dejar tu sistema funcionando a la perfección.
